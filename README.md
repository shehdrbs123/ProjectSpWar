# ProjectSpWar
 대학교 졸업과제(안드로이드 4인 네트워크 독립형 슈팅게임)(Unity Ver 2018.4.7f1)
 
 최대 4인까지 접속가능.
 로컬 네트워킹(와이파이) 아이피 접근을 통해서 게임진행
 
 구현내용
 - 로컬네트워킹(아이피/포트 입력)
 - 로비 동기화 및 진행
 - 인 게임 진행
   - 일반몹 2기 (각각 1개패턴)
   - 보스몹 1기 (4개패턴)
 - 게임 진행상황 동기화, 게임 종료 후 로비복귀 동기화
 
 미구현 내용
  - 차원간 서포트 시스템(다른 유저에게 스킬 써주기)
  - 스킬/ 스킬 UI
